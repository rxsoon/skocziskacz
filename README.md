# 🎮 Jump & Jump

A simple yet addictive **clicker / idle game** where timing, upgrades, and progression matter.

🌐 **Play now:** https://skocziskacz.pl

---

## 🚀 About the Game

You control a bouncing ball — every click makes it jump.
Every jump earns points. Every point pushes you forward.

Upgrade, optimize, and climb the global leaderboard 🔥

---

## ⚙️ Features

* 🟡 **Clicker gameplay** — click anywhere to jump and earn points
* ⚡ **Upgrades:**

  * x2 / x3 multipliers
  * Auto jump
  * Turbo mode
* 🔁 **Rebirth system** — reset progress for permanent boosts
* 🏆 **Live leaderboard (TOP 10)** — compete with other players in real time
* 👤 **User accounts (Firebase Auth)**
* ☁️ **Cloud save (Firestore database)**

---

## 🧠 How to Play

1. Click anywhere to jump
2. Earn points
3. Buy upgrades
4. Unlock rebirth for faster progression
5. Reach the TOP 10 leaderboard 💀
